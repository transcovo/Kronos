# Material DateTime Picker - Select a time/date in style

This project is forked from great android library made by Wouter Dullaert and is modified to allow these functionalities:
 - embed a material date time picker inside a layout
 - make it react without using ok and cancel buttons
 - remove seconds handling
 - make all colors customizable and remove preset colors
